########################
####para rodar o app####
########################
1.      instalar node: https://nodejs.org/en/download/
2.      abrir terminal
3.      > npm install --save-dev sequelize-cli
4.      > npx sequelize-cli db:migrate
5.      > npm install
6.      > node_modules/.bin/sequelize db:migrate
7.      > npm run dev
